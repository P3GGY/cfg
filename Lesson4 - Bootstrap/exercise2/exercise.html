Exercise

Make a grid that has 5 rows

Row 1: First column = 8 units, Second column = 4 units

Row 2: Three columns of equal width

Row 3: First column = 7 units, Second column = empty

Row 4: First column = empty, Second column = 8 units, Third column = empty

Row 5: Four columns of equal length, but First and Third columns are empty

Each column should contain the number relating to its width, leave empty columns blank

Put the number in a p tag inside the relevant columns. Use the Bootstrap class name "text-center" in the p tag to make the text aligned in the centre.
