ID

Sometimes you may only want to change one specific 'p' element without effecting any other 'p' element. One way to do this is by using inline css, but this is not advised. So the other way is to give an ID to the element you want to change.

For example, this paragraph has an ID called "blue-paragraph". Now we can set the css for the "blue-paragraph" ID and no other element on this page will be effected.

How to style an ID

Remember when you styled an element you just used the element name? Well, when styling a specific ID, use the '#' symbol before the ID name. e.g. '#blue-paragraph'.

When to use an ID

An ID must be unique. This means you cannot repeat the same ID name for a different element. So use an ID when you intend to apply CSS rules to only one specific element.

Class

There are occasions when you may want to apply the same CSS rules to different elements. In this case, use a class.

For example, every div on this page has a Class called "section". Now we can set the css for the "section" Class and only the elements with this class name will be effected.

How to style a Class

When styling a Class, use the '.' symbol before the Class name. e.g. '.section'.

When to use a Class

Use Classes when you want to set the same styles to multiple elements. Classes are useful to ensure that your website has consistent styles throughout. You only need to set the css rules once and then just add the class name to any element you want to style.
