So far the websites we've made are pretty boring :(

This is a demonstration of how we can (slightly) improve the layout of our page.

There are quite a few divs on this page. And there are divs inside other divs. The idea of divs is that they are a way of grouping related elements. Divs act as containers for other elements.

There's a fixed navigation menu bar ("navbar") at the top. If you scroll down, the navbar remains fixed at the top of the window.

To make a navbar you need to use an unordered list and set the following CSS rules on the list items:

  • list-style-type: none;
  • display: inline-block;
